# KickerConnect

KickerConnect is a mobile application designed for football enthusiasts to connect, organize games, and foster a community centered around the sport. Built using modern technologies, it bridges the gap between individuals looking to play and enjoy football.

## Features

- **User Registration and Login**: Users must create an account to access the application. Unique usernames and secure authentication ensure a personalized experience.
- **Group Events**: Centralized event management for organizing matches within groups.
- **User-Friendly Interface**: Streamlined design for effortless navigation and interaction.

## Technologies Used

- **Frontend**: React Native
- **Backend**: Spring Boot with Java
- **Database**: MySQL
- **Development Tools**: IntelliJ IDEA and Visual Studio Code

## Setup Instructions

1. Clone the repository:
   ```bash
   git clone https://github.com/AlexandruDobos/kickerconnect.git
   ```
2. Navigate to the `frontend` directory and run:
   ```bash
   npm install
   npm start
   ```
3. Navigate to the `backend` directory and start the server:
   ```bash
   mvn spring-boot:run
   ```
